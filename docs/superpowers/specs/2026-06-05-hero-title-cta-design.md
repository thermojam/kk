# Hero Title and CTA — design

## Цель

Улучшить читаемость Hero за счёт спокойного широкого заголовка и сократить тексты двух CTA.

## Изменения

- На desktop заголовок занимает более широкий текстовый блок и визуально складывается в две смысловые строки:
  - «Психология женского тела»
  - «и проявленности.»
- Перенос между смысловыми частями фиксируется только на `lg+`.
- На mobile заголовок переносится естественно, без принудительных строк.
- Размер desktop-заголовка немного уменьшается относительно текущих `72px`, чтобы первая строка помещалась целиком.
- Primary CTA меняется на иконку Telegram и текст «Написать».
- Secondary CTA меняется на «Мои услуги».

## Реализация

Изменяется только `components/sections/Hero.tsx`:

- заголовок разделяется на два inline-блока;
- перед второй смысловой частью добавляется desktop-only перенос;
- ширина текстовой колонки увеличивается;
- к Telegram CTA добавляется существующая иконка `Send` из `lucide-react`;
- тексты CTA сокращаются.

`TelegramButton`, общий `Button`, портрет, фон, анимации, ссылки и аналитика не меняются.

## Acceptance

- На desktop заголовок читается двумя устойчивыми смысловыми строками.
- На mobile заголовок переносится естественно и не уменьшается ради сохранения двух строк.
- Primary CTA показывает иконку Telegram и текст «Написать».
- Secondary CTA показывает текст «Мои услуги».
- Ссылки и аналитические цели CTA сохраняют текущее поведение.
- `npm run lint` и `npm run build` проходят.
