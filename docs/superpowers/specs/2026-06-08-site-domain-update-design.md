# Обновление адреса сайта

## Цель

Заменить устаревший адрес сайта `https://kamenskaya.ru` на реальный адрес
`https://ksenia-kamenskaya.ru` в актуальном содержимом и SEO-метаданных проекта.

## Объем изменения

- Обновить `BUSINESS.siteUrl` в `content/legal/business.ts`. Это изменит адрес на
  страницах политики обработки персональных данных, политики куки, публичной
  оферты и в JSON-LD.
- Обновить `metadataBase` и `openGraph.url` в `app/layout.tsx`.
- Не менять адрес электронной почты `hello@kamenskaya.ru`.
- Не менять исторические спецификации и планы в `docs/`.

## Проверка

- Поиск по актуальному коду не находит URL `https://kamenskaya.ru`.
- Поиск подтверждает использование `https://ksenia-kamenskaya.ru` в общей
  бизнес-константе и SEO-метаданных.
- `npm run lint` и `npm run build` завершаются успешно.
