# Мобильные CTA-кнопки как в Hero

## Цель

Сделать кнопки в `Offer` и `ContactCtaBanner` во всю доступную ширину и высотой
`52px` на мобильных экранах, повторив размеры кнопок в `Hero`.

## Изменение

- Добавить `w-full h-[52px] sm:h-11 sm:w-auto` кнопке в
  `components/sections/Offer.tsx`, сохранив существующий верхний отступ.
- Добавить `w-full h-[52px] sm:h-11 sm:w-auto` кнопке в
  `components/sections/ContactCtaBanner.tsx`.
- Не менять общий компонент `Button` и desktop-верстку.

## Проверка

- На ширине меньше `640px` обе кнопки занимают доступную ширину контейнера и
  имеют высоту `52px`.
- Начиная с `640px` обе кнопки имеют автоматическую ширину и текущую высоту
  `44px`.
- `npm run lint` и `npm run build` завершаются успешно.
