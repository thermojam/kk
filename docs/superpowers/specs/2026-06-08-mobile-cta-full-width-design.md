# Мобильные CTA-кнопки во всю ширину

## Цель

Сделать кнопки в `Offer` и `ContactCtaBanner` во всю доступную ширину на
мобильных экранах, повторив responsive-поведение кнопок в `Hero`.

## Изменение

- Добавить `w-full sm:w-auto` кнопке в `components/sections/Offer.tsx`,
  сохранив существующий верхний отступ.
- Добавить `w-full sm:w-auto` кнопке в
  `components/sections/ContactCtaBanner.tsx`.
- Не менять общий компонент `Button` и desktop-верстку.

## Проверка

- На ширине меньше `640px` обе кнопки занимают доступную ширину контейнера.
- Начиная с `640px` обе кнопки имеют автоматическую ширину.
- `npm run lint` и `npm run build` завершаются успешно.
